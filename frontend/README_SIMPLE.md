# Frontend

Electron + React UI for the transcription app.

## Setup
```bash
npm install
```

## Run
```bash
npm run dev     # Development
npm run build   # Production build
npm run dist    # Create installer
```

## Main Components

- `App.tsx` - Main application
- `components/UploadTab.tsx` - File upload
- `components/TranscribeTab.tsx` - Transcription UI
- `components/GlobalFileSelector.tsx` - File management

## Tech Stack
- Electron (desktop wrapper)
- React + TypeScript
- Tailwind CSS
- Vite
